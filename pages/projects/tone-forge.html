toneforge

$ a text lingo conversion REST API

overview

ToneForge is a public REST API that converts text into specific lingos using AI. For example, using the Pirate lingo: Hello, Friend → Ahoy Matey! The conversion is handled by the Gemini API, so results are natural and contextual rather than a word-swap.

Rate limits are strict due to free-tier constraints on both Gemini and AWS EC2. Started as a personal experiment, made public for anyone to build on.

tech stack

  • Java — Spring Boot
  • AWS EC2 — hosting
  • Google Gemini API — language conversion

links

GitHub Live API

endpoints

Two open endpoints, no authentication required:

  • GET /api/strategies — returns a list of all available lingos
  • POST /api/convert — converts text to the specified lingo

usage

Send a POST to http://52.207.146.61/api/convert with a JSON body:

request

{
    "strategy": "Pirate",
    "text": "Hello, Friend"
}

response

{
    "original": "Hello, Friend",
    "converted": "Ahoy Matey!",
    "lingo": "Pirate",
    "error": null
}

javascript example

const response = await fetch(
    `http://52.207.146.61/api/convert`,
    {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ strategy, text })
    }
);
