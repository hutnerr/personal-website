toneforge

$ a text lingo conversion api

Overview

ToneForge is a public REST API that converts text into specific lingos using AI. For example, using the Pirate lingo, it might convert: Hello, Friend → Ahoy Matey!

It is built with Spring Boot, hosted on AWS EC2, and uses the Gemini API for the language conversion. Rate limits are strict due to free-tier constraints on both Gemini and AWS.

The source code is available on GitHub.

Technologies Used

  • Java: Spring Boot
  • Cloud: AWS EC2
  • AI: Gemini API

Links

  • GitHub Repository
  • Live API

The Product

ToneForge exposes a simple API that takes a piece of text and a target lingo (called a "strategy") and returns the converted text. The conversion is handled by the Gemini API under the hood, so the results are natural and contextual rather than a simple word-swap.

The /strategies endpoint returns all available lingos, and the /convert endpoint does the conversion. Both are open and require no authentication.

It started as a personal experiment but ended up being something I thought others might want to play with or build on top of, so I made it public.

Endpoints

  • POST /api/convert - Convert text to the specified lingo
  • GET /api/strategies - Returns a list of all available lingos

Usage

Send a POST request to http://52.207.146.61/api/convert with a JSON body containing a strategy and text field.

Request

{
    "strategy": "Pirate",
    "text": "Hello, Friend"
}

Response

{
    "original": "Hello, Friend",
    "converted": "Ahoy Matey!",
    "lingo": "Pirate",
    "error": null
}

JavaScript Example

const response = await fetch(
    `http://52.207.146.61/api/convert`,
    {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json'
        },
        body: JSON.stringify({
            strategy,
            text
        })
    }
);
