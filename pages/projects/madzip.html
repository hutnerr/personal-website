madzip

$ a file compression utility

overview

MadZip is a file compression utility that uses a custom Huffman coding implementation to compress and decompress files. It features a JavaFX GUI for file selection and shows compression statistics after each operation.

tech stack

  • Java — JavaFX for the GUI
  • Data structures — priority queues, binary trees
  • Algorithms — Huffman coding

GitHub

showcase

takeaways

Implementing Huffman coding from scratch was a great exercise in data structures and algorithm design — using priority queues to build the frequency tree and then encoding/decoding with the resulting bit strings. Some key takeaways:

  • Deep understanding of Huffman coding and lossless compression
  • Priority queue and binary tree implementation in practice
  • Bit-level I/O and file format design
