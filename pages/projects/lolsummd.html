lolsummd

$ a champion mastery aggregator for League of Legends

overview

LOLSUMMD aggregates League of Legends champion mastery data across multiple accounts. Add up to 10 accounts by Name#Tag, and the app fetches and combines mastery points and levels for every champion - useful if your history is split across accounts.

API responses are cached in Redis via Upstash to reduce calls to Riot's servers. The app is deployed on Vercel with a Flask backend. Champions can be sorted by name, mastery points, or levels.

tech stack

  • Python - Flask backend
  • Redis via Upstash - API response caching
  • Riot Games API - champion mastery data
  • HTML/CSS/JavaScript - frontend
  • Vercel - hosting
