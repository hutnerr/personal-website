pantry panic

$ a recipe finder for ingredients you already have

overview

A group web development project. Pantry Panic lets you manage a virtual "pantry" of ingredients you have on hand, then searches the Spoonacular API for recipes you can actually make with them. My primary role was backend integration — wiring up the recipe API and managing data flow.

Note: the project is currently non-functional due to the Spoonacular API service being cancelled. The demo links to a personal fork.

tech stack

  • HTML, CSS, JavaScript — frontend
  • Spoonacular API — recipe search
  • Local storage — saved favorites and pantry persistence
  • Git & GitHub — version control

takeaways

First time integrating a third-party REST API — it taught a lot about async request handling, error states, and the gap between API documentation and production behavior. Some key takeaways:

  • REST API integration and async JavaScript
  • Group coordination — communication prevents most merge conflicts and missed dependencies
  • User experience as a first-class concern in frontend design
