tip calculator

$ an ad-free Android tip calculator

overview

A simple, ad-free Android tip calculator built to get started with Android development. Open the app, type the bill amount, and the tip is calculated immediately using your saved settings - no extra taps. Features persist between sessions so it's always configured to your preference.

Supports rounding modes, multiple currencies, bill splitting among up to 99 people, and several preset themes. Works entirely offline.

tech stack

  • Java - Android SDK, OOP
  • Android Studio - Gradle build system
