lospec daily bot

$ a pixel art daily challenge Discord bot

overview

A Discord bot that scrapes and posts the daily pixel art challenges from Lospec into any server channel. It's meant to provide daily inspiration and a prompt for pixel artists to practice.

Originally built for personal use, it was made public once it felt polished enough that others might find it useful too. Not affiliated with Lospec.

tech stack

  • Python — Discord.py, Aiohttp, BeautifulSoup4
