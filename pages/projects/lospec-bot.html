01.

Overview

This is a Discord Bot created using Discord.py. It posts the daily pixel art challenges from Lospec into a Discord server channel daily. It is meant to provide some inspiration and opportunity for pixel artists to practice their skills.

As a live service product, it is a constant work in progress and always changing. The source code is available on GitHub.

Technologies Used

  • Python: Discord.py, Aiohttp, BS4

02.

The Product

The final product is a quite simple Discord bot that posts the daily pixel art challenges from Lospec.

I was originally going to just create a static bot that only worked in my own server, but I figured that other people might find it useful as well, so I made it public and made it available for anyone to add to their own servers.

Overall, it is a simple bot, but it serves its purpose well and is a fun little project to work on.

One other thing to note is that this bot is not affiliated with Lospec in any way, it is simply a fan project.
