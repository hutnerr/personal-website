tough cents

$ a behavior-driven financial accountability tool

overview

Tough Cents is a web app that uses "tough love" behavioral psychology to discourage poor spending. Scan a receipt and Google Gemini reads the purchase details; Ollama then generates a personalized, sometimes harsh commentary based on your spending history. Because adults need accountability too.

You can also link a bank account for direct transaction data, enabling more comprehensive tracking and feedback over time. The app gamifies financial discipline with a points system and push notifications.

tech stack

  • Next.js - frontend
  • Flask - backend API
  • PostgreSQL - 14+ table relational database
  • Ollama - AI commentary generation
  • Google Gemini - receipt OCR and parsing

showcase

takeaways

The most challenging and rewarding part was the database design - 14+ interconnected tables covering users, transactions, receipts, categories, and feedback history. Getting the schema right upfront made everything downstream easier. Some key takeaways:

  • Complex relational database design - schema modeling and normalization
  • Using multiple AI models for distinct purposes in one app
  • Full-stack development across Next.js, Flask, and PostgreSQL
