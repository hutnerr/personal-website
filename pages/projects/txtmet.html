txtmet

$ a command-line text metric utility

overview

TXTMET is a lightweight CLI tool that reads a file and outputs word count, character count, line count, or sentence count. Born out of frustration with tools that are either overkill or not flexible enough for simple counting tasks.

Implemented in both C and Rust. Flags are handled via getopt in the C version.

tech stack

  • C — standard library, file I/O, getopt
  • Rust — alternate implementation

links

GitHub

takeaways

Implementing the same utility in two languages side by side is a useful exercise — Rust's ownership model and pattern matching made certain parts significantly cleaner than the C equivalent. Some key takeaways:

  • Hands-on C and Rust experience with real file I/O
  • Command-line argument parsing with getopt
  • Comparing idiomatic approaches across languages
