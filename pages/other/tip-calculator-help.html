01.

Overview

The Basics

The tip calculator is quite simple and streamlined to use. It has various features to make it appropriate for every day use (see below).

To use it, simply start typing in the bill amount, and it will be calculated as you type based on the current configuration (tip percentage, currency, rounding, etc.). You can adjust these settings in the settings menu.

From the home page, there exists two buttons on the top title bar. The one that looks like a little person toggles the "Splitting" menu where you can split the bill among multiple people. The gear icon opens the settings menu where you can adjust various settings of the app.

Your settings will persist which makes it incredibly easy to simply open up the app, type in the bill amount, and have the tip calculated (and/or split) in seconds.

Features

  • Rounding System: Users can choose how/if the tip is rounded
  • Customizable: Users can set up the app to their preferences
  • Currency Support: Includes a few popular currencies
  • Themes: Multiple preset themes available
  • Default Tip Percentage: Set your preferred default tip
  • Bill Splitting: Split the total evenly among up to 99 people
  • Ad-free: Enjoy an ad-free experience for uninterrupted use
  • Offline: The app can be used entirely offline
02.

Settings

Settings Walkthrough

The settings menu allows you to customize various aspects of the tip calculator to your liking. Here are the available settings:

Theme: Choose the visual theme of the app. Options include Gruvbox, Dracula, and Mocha.

Rounding Method: Choose how the total amount is rounded. Options include:

  • No Rounding: The total is not rounded. (5.49 remains 5.49)
  • Dynamic Rounding: The total amount is rounded to the closest whole number (5.49 -> 5.00 or 5.51 -> 6.00).
  • Always Down: The total amount is always rounded down to the nearest whole number (5.99 -> 5.00 or 5.15 -> 5.00).
  • Always Up: The total amount is always rounded up to the nearest whole number (5.49 -> 6.00 or 5.15 -> 6.00).

Currency: Select the currency symbol used in the app. Options include USD ($), GBP (£), EUR (€), INR (₹), PHP (₱), THB (฿), or None.

Default Tip Percentage: Set the tip percentage that is used by default when opening the app.

Max Split: Set the maximum number of people that can be split among (up to 99).

Similar to the previous page, there also exists buttons on the top title bar for navigation.

  • The button that looks like a arrow pointing to itself is a button to reset all settings to their defaults.
  • The X button closes the settings menu and takes you back to the main calculator.
